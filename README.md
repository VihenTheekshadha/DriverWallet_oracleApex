# Digital Wallet for Driver’s License & Insurance

## Overview

This project is a web-based digital wallet application built entirely using Oracle APEX. The application allows users to securely store and access verified digital versions of their Driver’s License and Vehicle Insurance documents.

The idea behind this application came from a real-life experience where I forgot to bring my physical driver’s license and insurance while traveling. This resulted in spending several minutes stopped by police officers until a family member brought the documents to avoid receiving a fine.

The purpose of this solution is to eliminate the need to carry physical copies or unreliable photos of important documents by providing a secure and verified digital alternative that can be accessed anytime through a mobile device.

---

## Problem Statement

Many drivers forget to carry:
- Driver’s License
- Vehicle Insurance
- Other important driving documents

People often take photos of these documents on their phones, but:
- Photos can be edited or forged
- There is no verification process
- Security and authenticity are questionable

This application solves that issue by allowing only verified documents to be stored and displayed inside the system.

---

## Solution

The system provides:
- Secure user authentication
- Verified digital storage of documents
- Admin approval workflow
- QR code generation for quick access
- Mobile-friendly access to documents

Users can log in using their username and password and instantly access their verified documents from anywhere.

---

## Features

### User Features
- User registration and login
- Upload Driver’s License
- Upload Vehicle Insurance
- QR code generation
- Download/save QR code to phone
- Mobile-friendly interface
- Quick document access

### Admin Features
- Review uploaded documents
- Verify Driver’s License
- Verify Insurance
- Approve or reject submissions
- Maintain document authenticity

---

## How It Works

1. User creates an account
2. User uploads Driver’s License and Insurance
3. Admin reviews submitted documents
4. Approved documents become accessible in the wallet
5. User generates a QR code
6. QR code can be saved to the phone for fast access
7. Scanning or opening the QR provides immediate access to verified documents
